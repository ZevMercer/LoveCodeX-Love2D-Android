**Love CodeX** adalah editor kode Lua untuk **LÖVE (Love2D)** di Android.  
Aplikasi ini dibuat untuk memudahkan pembuatan proyek game LÖVE langsung dari HP: menulis script, mengelola file proyek, mengimpor aset, lalu mengekspornya menjadi file **`.love`** yang siap dijalankan.

## Apa itu Love CodeX?
Love CodeX adalah editor dan pembangun proyek **LÖVE (Love2D)** berbasis Android.  
Tujuannya adalah menghadirkan workflow pengembangan game LÖVE yang ringan, praktis, dan mobile-friendly tanpa harus selalu bergantung pada PC.

## Tujuan Love CodeX
Love CodeX dibuat untuk memudahkan pembuatan proyek **LÖVE (Love2D)** di Android, terutama dalam proses menghasilkan file **`.love`** tanpa harus repot membungkus proyek secara manual seperti saat memakai editor biasa.

Dengan Love CodeX, pengguna dapat menulis script Lua, mengelola file proyek, mengimpor aset, lalu mengekspor semuanya langsung menjadi file **`.love`** yang siap dijalankan di **LÖVE for Android / LÖVE Loader**.

Ke depannya, Love CodeX dirancang untuk berkembang menjadi workflow LÖVE mobile yang lebih praktis, termasuk kemungkinan menjalankan hasil proyek langsung dari Love CodeX dengan membuka runtime **LÖVE** secara otomatis.

## Fitur Utama
- Buat dan kelola proyek LÖVE
- Edit file `.lua`
- File explorer untuk folder dan file proyek
- Import aset gambar dan audio
- Export project menjadi file **`.love`**
- Kompatibel dengan **LÖVE Loader / LÖVE for Android**
- UI dark mode ringan dan fokus untuk mobile

## Workflow
1. Buat proyek baru
2. Tambah file atau folder
3. Tulis script Lua
4. Import aset gambar / audio
5. Export ke file `.love`
6. Jalankan di **LÖVE Loader / LÖVE for Android**

## Status Project
Love CodeX saat ini masih berada dalam tahap **Alpha**, tetapi sudah dapat digunakan untuk workflow dasar pembuatan project LÖVE di Android.

## Roadmap
Fokus pengembangan berikutnya:
- multitab editor yang persisten
- autosave yang lebih solid
- restore state File Explorer
- scroll editor full-area
- manajemen file export `.love`
- kemungkinan menjalankan project langsung dari Love CodeX

## Catatan
Love CodeX adalah **editor dan project builder**, bukan runtime game.  
Untuk menjalankan file hasil export, gunakan **LÖVE for Android / LÖVE Loader**.
